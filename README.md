# new-agent-memory

## 项目简介

这是一个探索**类人记忆系统**的实验性项目。

核心理念：
- **越用越聪明** — 系统从使用中自适应调整，而不是一蹴而就
- **遗忘是 feature 不是 bug** — 长期不用的记忆权重自然衰减，是进化结果
- **知识有边界、有成长过程** — 像树苗慢慢长出年轮，不应该像压缩包瞬间解压

## 核心设计

```
主体（索引）+ 切片（本地存储）+ 重建
```

- **多维索引**：时间/人物/地点/情绪/主题 多维度靶向检索
- **碎片存储**：记忆切片化，本地持久化
- **动态重建**：根据上下文触发重建，而非简单读取

## 灵感来源

- 大脑皮层四语言区模块化思路（W 听说读写 / VI 觉 / H 听觉 / V 视觉）
- 向量检索作为补充手段
