# 训练指南：0.1B 记忆增强模型

## 🎯 目标

训练一个 **0.1B (100M) 参数**的记忆增强模型，包括：
- 记忆编码器 (~10M)
- 记忆检索器 (~5M)
- 记忆适配器 (~85M)

---

## 📊 模型架构

```
输入文本
    ↓
┌─────────────────────────────────────────────────────────────┐
│ 记忆编码器 (MemoryEncoder) - 10M 参数                        │
│   - 6层 Transformer                                          │
│   - 隐藏维度: 256                                            │
│   - 注意力头: 4                                              │
│   - 输出: 记忆向量 (128维)                                    │
└─────────────────────────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────────────────────────┐
│ 记忆检索器 (MemoryRetriever) - 5M 参数                       │
│   - 双塔结构                                                  │
│   - 查询编码器 + 记忆编码器                                    │
│   - 输出: 相关性分数                                          │
└─────────────────────────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────────────────────────┐
│ 记忆适配器 (MemoryAdapter) - 85M 参数                        │
│   - 12层 Transformer                                         │
│   - 隐藏维度: 512                                            │
│   - 注意力头: 8                                              │
│   - 输出: 格式化的记忆表示                                     │
└─────────────────────────────────────────────────────────────┘
    ↓
现有 LLM (GPT/LLaMA)
```

## 📚 训练数据

### 数据来源
1. **对话数据集**：MultiWOZ, PersonaChat, TopicalChat
2. **记忆数据集**：自建对话记忆对
3. **预训练数据**：Common Crawl 子集

### 数据格式
```json
{
  "conversation": [
    {"role": "user", "content": "我叫张三"},
    {"role": "assistant", "content": "你好张三！"}
  ],
  "memory": {
    "type": "episodic",
    "content": "用户叫张三",
    "importance": 0.8,
    "emotion": "joy"
  }
}
```

## 🛠️ 训练步骤

### Step 1: 环境准备
```bash
# 创建虚拟环境
conda create -n memory-model python=3.10
conda activate memory-model

# 安装依赖
pip install torch transformers datasets accelerate wandb tqdm
```

### Step 2: 准备数据
```bash
python data_generator.py
```

### Step 3: 训练模型（自动化 - 推荐）
```bash
# 使用自动化训练智能体（一键启动，无需人工干预）
python auto_trainer.py

# 或者使用标准训练
python train.py
```

### Step 4: 评估模型
```bash
python evaluate.py
```
